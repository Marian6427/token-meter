# 🔥 Token Meter

**Task manager for your AI spend.** A floating widget that shows real-time token consumption per third-party app in AI chat interfaces.

## What it does

- Shows which app is burning how many tokens — right now
- Estimates session cost and monthly projection
- Tells you if API pay-per-token would be cheaper than your flat plan

## Why

You pay $200/month. An agent silently burns $30 per task. You don't know until it's too late.

**You can't optimize what you can't measure.**

## Quick start

```bash
npm install && npm run dev
```
